A multi-role Student Management System built using Core PHP and MySQL.

## Features

- Role-based authentication (Admin, Teacher, Student)
- Admin dashboard
- Teacher management
- Subject management
- Teacher-Subject assignment
- Student registration
- Marks management
- Syllabus viewing
- Clean UI with modular includes

## Tech Stack

- Core PHP
- MySQL
- HTML5
- CSS3
- Bootstrap
- Git & GitHub

## Modules

### Admin
- Manage teachers
- Manage subjects
- Assign subjects to teachers

### Teacher
- View assigned subjects
- Enter marks
- View syllabus

### Student
- Register
- View marks
- View syllabus

## Database Tables

- users
- students
- teachers
- subjects
- teacher_subjects
- marks

## Setup

1. Import database.sql
2. Configure connection.php
3. Run using:

php -S localhost:8000 -t public
